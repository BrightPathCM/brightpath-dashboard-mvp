# Bright Path Dashboard Prototype

A polished React + TypeScript dashboard prototype for Bright Path, a SaaS-style operational intelligence product for dental practices.

The app is structured as a Vite + React + TypeScript project and is ready for Vercel deployment.

## Local Development

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
```

Vercel settings:

- Build Command: `npm run build`
- Output Directory: `dist`

## Included

- Full app shell with dark navy sidebar and top header
- Executive Overview dashboard with KPI cards, health score, charts, provider table, recommendations, and alerts
- Sidebar navigation for all requested pages, with placeholder page surfaces
- Reusable TSX components and mock-only data
